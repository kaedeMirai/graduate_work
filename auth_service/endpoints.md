### Users API

    Endpoints:
        - signup
        - login
        - logout
        - login_history
        - reset_login
        - reset_password
        - refresh

### Roles API

    Endpoints:
        * CRUD:
            - create_role
            - roles (get all existing roles)
            - update_role
            - delete_role
        - add_role (add role to user)
        - remove_role (remove user role)
        - has_role (check if user has specified role)
